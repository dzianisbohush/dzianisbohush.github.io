Moon War

Protect the moon from alien invaders!

START GAME

CONTROLS

Use or W S A D for running and SHIFT for inputting tower

BONUSES

Collect blue crystals for increasing count hero lifes, green for increasing bullets speed and red for killing all enemies

TECNOLOGIES

JavaScript, HTML5 Canvas, Bootstrap, Sass, ESLint (Airbnb style), Webpack

CONTROLS

or button "W" - run up

or button "S" - run down

or button "A" - run left

or button "D" - run right

button "SHIFT" - input tower

button "P" - pause

button "M" - mute sounds

BONUSES

Blue crystals - increasing count hero lifes and adding 10000 to score

Green crystals - increasing bullets speed and adding 2000 to score

Red crystals - killing all enemies and adding 5000 to score
